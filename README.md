# Task Allocation for Front-end Developer

## Task 1: Create a Landing Page
**Description:** Create an attractive landing page for a website with eye-catching visuals, easy-to-find buttons, and interesting content to grab visitors' attention. Make sure it looks good on phones and computers, so everyone can enjoy it. The goal is to get people excited and eager to see more of what the website has to offer.

**Files:**
- `index.html`: HTML file for the landing page structure.
- `style.css`: CSS file for styling the landing page.
- `script.js`: JavaScript file for adding interactivity to the landing page.

**How to Open Files:** 
To view/edit HTML, CSS, and JavaScript files:
1. Open `landing_page.html` in a web browser to see the landing page layout.
2. Use a text editor (e.g., Visual Studio Code, Sublime Text) to open `landing_page.html`, `landing_page.css`, and `landing_page.js` for editing.

**Tech Stack:**
- HTML
- CSS
- JavaScript

## Task 2: Create a Calculator
**Description:** Develop a simple calculator web application that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. Design a clean and user-friendly interface with buttons for each digit and operation, ensuring easy input and clear display of results. Implement functionality to handle calculations accurately and provide instant feedback to users.

**Files:**
- `index.html`: HTML file for the calculator interface.
- `style.css`: CSS file for styling the calculator.
- `script.js`: JavaScript file for implementing calculator logic.

**How to Open Files:** 
To view/edit HTML, CSS, and JavaScript files:
1. Open `calculator.html` in a web browser to use the calculator.
2. Use a text editor to open `calculator.html`, `calculator.css`, and `calculator.js` for editing.

**Tech Stack:**
- HTML
- CSS
- JavaScript

## Task 3: Create a To-Do List Application
**Description:** Develop a simple yet effective To-Do List Application where users can easily add, edit, and delete tasks. The application should feature an intuitive user interface with options to mark tasks as completed and organize them by priority or due date. Users should also be able to view their tasks in a clear and organized manner, ensuring seamless task management and productivity.

**Files:**
- `index.html`: HTML file for the to-do list interface.
- `styles.css`: CSS file for styling the to-do list.
- `script.js`: JavaScript file for implementing to-do list functionality.

**How to Open Files:** 
To view/edit HTML, CSS, and JavaScript files:
1. Open `todo_list.html` in a web browser to use the to-do list application.
2. Use a text editor to open `todo_list.html`, `todo_list.css`, and `todo_list.js` for editing.

**Tech Stack:**
- HTML
- CSS
- JavaScript
